## 🚨 TROUBLESHOOTING

### App won't start?
```bash
npm install
npm start
```

### Can't scan QR code?
- Check WiFi (phone and laptop must be on same network)
- Try `npm run web` to test in browser

### Picker component error?
```bash
npm install @react-native-picker/picker @react-native-community/slider
```

### Need to reset?
```bash
expo start -c
```
